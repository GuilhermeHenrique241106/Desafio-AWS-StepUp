Desafio de projeto focado na consolidação de conhecimentos práticos sobre workflows automatizados com o AWS Step Functions. O objetivo principal foi aplicar os conceitos de orquestração serverless apresentados em aula e documentar de forma clara e estruturada toda a experiência técnica obtida durante a prática. Como entregável, foi criado um repositório público organizado no GitHub contendo um arquivo README.md detalhado com anotações, insights adquiridos e capturas de tela que servem como evidência de execução da infraestrutura e material de apoio para futuras implementações.
